Privacy Policy for Learn to Speak

Learn to Speak is a reading practice application that uses Text-To-Speech (TTS) and Speech-To-Text (STT) features to help users improve pronunciation while reading books.

Information Collection and Use

The Application does NOT collect or transmit any personal data to external servers.

  • Audio input from the microphone is used only for real-time pronunciation analysis (STT) and is NOT stored.
  • Reading progress and history are stored locally on the user's device using a mobile database.
  • No login, account, or personally identifiable information is required to use the app.

Permissions Used

  • Microphone: Used for speech recognition and pronunciation checking.
  • Speaker: Used for Text-To-Speech book reading.

Data Storage

All data related to reading progress is stored locally on the device. The Service Provider does not have access to this data.

Third Party Access

This application does not share any data with third parties and does not use analytics, advertising SDKs, or tracking services.

User Rights

Users can delete all stored data at any time by uninstalling the application.

Children's Privacy

This application does not knowingly collect personal information from children.

Security

Since no personal data is transmitted or stored externally, user privacy is inherently protected.
